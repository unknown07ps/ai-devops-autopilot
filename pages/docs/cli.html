CLI Tool

Command-line interface for Deployr

Installation

# Install via pip
pip install deployr-cli

# Or download binary from GitHub releases
curl -L https://github.com/unknown07ps/deployr-cli/releases/latest/download/deployr-linux-amd64 -o deployr
chmod +x deployr
sudo mv deployr /usr/local/bin/

Configuration

# Configure API endpoint and token
deployr config set api-url http://localhost:8000
deployr config set api-token your-api-token

# Verify configuration
deployr config show

Commands

Incidents

Command Description
deployr incidents list List all open incidents
deployr incidents get {id} Get incident details
deployr incidents ack {id} Acknowledge incident
deployr incidents resolve {id} Resolve incident

Actions

Command Description
deployr actions pending List pending approvals
deployr actions approve {id} Approve action
deployr actions reject {id} Reject action
deployr actions history Show action history

Status

Command Description
deployr status Quick health check
deployr status detailed Component-level status
deployr metrics Current system metrics

Examples

# List recent incidents with severity
deployr incidents list --limit 10 --severity critical

# Approve all pending actions for a service
deployr actions pending --service api-gateway | deployr actions approve --stdin

# Watch for new incidents in real-time
deployr incidents watch
