API Reference

RESTful API for Deployr integration

Base URL: http://localhost:8000/api/v1

Authentication

All API requests require a Bearer token in the Authorization header.

Authorization: Bearer <your-api-token>

Health & Status

GET /health System health check
GET /health/detailed Detailed component status

Incidents

GET /incidents List all incidents
GET /incidents/{id} Get incident details
POST /incidents/{id}/acknowledge Acknowledge incident
POST /incidents/{id}/resolve Resolve incident

Actions

GET /actions/pending List pending actions
POST /actions/{id}/approve Approve action
POST /actions/{id}/reject Reject action
GET /actions/history Action execution history

Data Ingestion

POST /ingest/metrics Ingest Prometheus metrics
POST /ingest/logs Ingest application logs
POST /ingest/deployment Register deployment event

Configuration

GET /config/auto-approve Get auto-approve settings
PUT /config/auto-approve Update auto-approve settings
GET /config/thresholds Get detection thresholds
